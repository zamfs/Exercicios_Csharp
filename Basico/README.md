# Basic C# Exercises

This repository contains basic C# exercises focused on learning the language and becoming familiar with the Visual Studio environment.

The projects were built using console applications, covering fundamental programming concepts and the basics of C# development.

## Goals

- Understand basic C# syntax
- Learn the structure of a .NET application
- Practice programming logic
- Work with console input and output
- Use conditional statements and loops
- Explore how Visual Studio works
- Become familiar with project and file organization

## Topics Covered

- Variables and data types
- Mathematical and logical operators
- Conditional statements (`if`, `else`, `switch`)
- Loop structures (`for`, `while`, `foreach`)
- Methods
- Arrays and lists
- User input with `Console.ReadLine()`
- Console output with `Console.WriteLine()`
- Random number generation
- Basic null value handling

## Technologies Used

- C#
- .NET
- Visual Studio
